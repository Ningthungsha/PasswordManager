# 🔐 Secure Password Manager

A Python-based password manager with **military-grade encryption** and password strength checking.

## ✨ Features

- 🔒 **AES Encryption** - Passwords encrypted securely
- 📊 **Password Strength Checker** - Rates passwords and gives feedback
- 💾 **Secure Storage** - Encrypted passwords stored locally
- 🔑 **Encryption Key** - Unique key for security
- ➕ **Add Passwords** - Save passwords easily
- 🔍 **Get Passwords** - Retrieve saved passwords
- ✏️ **Update Passwords** - Change passwords anytime
- ❌ **Delete Passwords** - Remove passwords

## 📋 Requirements
- Python 3.7 or higher
- cryptography library
